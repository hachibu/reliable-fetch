# Contributing

## Feature Release Checklist

-   [ ] Add tests.
-   [ ] Write documentation.
-   [ ] Add code examples in the examples folder.
-   [ ] Update `README.md` features and roadmap.
-   [ ] Compile source code and documentation with `yarn build`.

## Publish NPM Package Checklist

-   [ ] `yarn login`
-   [ ] `yarn publish`
