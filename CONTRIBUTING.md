# Contributing

## Code Review Checklist

-   [ ] Add tests.
-   [ ] Add documentation.
-   [ ] Add code examples in the examples folder.
-   [ ] Update Features and Roadmap section in `README.md`.
-   [ ] Compile source code and documentation with `yarn build`.

## Publish NPM Package Checklist

-   [ ] `yarn login`
-   [ ] `yarn publish`
